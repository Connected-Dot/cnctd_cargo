# cnctd_cargo

Cargo/Rust toolchain wrapper for programmatic builds and operations.

## Installation

```toml
[dependencies]
cnctd_cargo = { git = "https://github.com/Connected-Dot/cnctd_crates.git" }
```

## Features

- Build projects programmatically
- Publish to registries
- Version management
- Workspace operations
